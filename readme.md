# Python OOP Practice

Python exercises focused on object-oriented programming, written to refresh my oop knowledge as a part of a 30 day self structured study plan to prepare for professional Python develompent.

## Content:

- "bank_account.py" A simplified Bank account and savings account system demonstrating classes, inheritance, properties, static methods, dunder methods, decorators, and error handling.
- "shopping_cart.py" A ShoppingCart class demonstrating list comprehensions and dunder methods.

## Concepts:

- Classes, instances, and constructers
- Inheritance
- Duner methods
- Properties with validation
- Static methods and class attributes
- Custom decorators
- List comprehensions
- Exception handling

## Running the code:
on Linux:
``` bash
git clone https://github.com/osameri/python-oop-practice
cd python-oop-practice
python3 bank_account.py
```
